# Class 12 Biology — Quiz Authoring Plan

Goal: a complete chapter-wise MCQ quiz bank for **Class 12 Biology** (Bihar Board / NCERT
pattern), 50–60 questions per chapter, plus a mixed Full Syllabus Test (auto-generated by
sampling published chapters).

- **Data file:** `data/bio12.ts` (each chapter = `{ slug, title, unit, questions[] }`)
- **UI:** `app/quiz/page.tsx` (hub + player), `components/Quiz.tsx` (player), styles in `app/globals.css`
- **Format:** answer all → submit → scorecard + per-question explanation. Open to everyone.
- ⚠️ Questions are AI-authored — **spot-check each chapter's answer key** before publishing as exam-ready.
